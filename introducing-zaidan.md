  🎉 Introducing Zaidan — ShadCN UI for SolidJS!

  After lots of work, I'm excited to share Zaidan, a fully-featured ShadCN UI registry built specifically for SolidJS! It brings 53 beautifully designed, accessible components — powered by Kobalte & Corvu — with a customizable
  design system (5 styles, 20+ color themes, multiple fonts & border radii). Get started with one CLI command using the official shadcn CLI!

  🚀 What's included:
  • 53 production-ready components
  • Works with shadcn CLI (bunx shadcn@latest init)
  • Tailwind CSS v4 with modern theming
  • Multi-framework support (Vite, Astro, TanStack Start, Solid Start)

  📦 Coming soon: RTL docs, Base-UI flavor, and Agents markdown!

  I hope Zaidan can serve as a solid foundation (that's what "Zaidan" means in Japanese! 🇯🇵) for more SolidJS registries from the community.

  Huge thanks to the amazing teams behind SolidJS, TanStack, Kobalte, Corvu, and Tailwind CSS — couldn't have built this without you all! 💙
