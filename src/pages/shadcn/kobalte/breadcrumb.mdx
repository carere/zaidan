# Breadcrumb

Displays the path to the current resource using a hierarchy of links.

## Installation

### CLI

```package-dlx
shadcn@latest add @zaidan/breadcrumb
```

### Manual

Copy and paste the following code into your project.

```tsx file=../../../registry/kobalte/ui/breadcrumb.tsx showLineNumbers

```
