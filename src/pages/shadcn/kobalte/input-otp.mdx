# Input OTP

Accessible one-time password component with copy paste functionality.

## Installation

### CLI

```package-dlx
shadcn@latest add @zaidan/input-otp
```

### Manual

Install the following dependencies:

```package-install
@corvu/otp-field
```

Copy and paste the following code into your project.

```tsx file=../../../registry/kobalte/ui/input-otp.tsx showLineNumbers

```
