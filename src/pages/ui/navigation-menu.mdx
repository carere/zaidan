# Navigation Menu

A collection of links for navigating websites.

## Installation

### CLI

```package-dlx
shadcn@latest add navigation-menu
```

### Manual

Copy and paste the following code into your project.

```tsx file=../../registry/ui/navigation-menu.tsx showLineNumbers

```

## Usage

```tsx
import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuIndicator,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuTrigger,
  navigationMenuTriggerStyle,
} from "~/components/ui/navigation-menu";
```

```tsx showLineNumbers
<NavigationMenu>
  <NavigationMenuItem>
    <NavigationMenuTrigger>Item One</NavigationMenuTrigger>
    <NavigationMenuContent>
      <NavigationMenuLink>Link</NavigationMenuLink>
    </NavigationMenuContent>
  </NavigationMenuItem>
</NavigationMenu>
```

## Link

You can use the `as` prop to make another component render as a navigation menu link.

```tsx showLineNumbers title="components/example-navigation-menu.tsx"
import { A } from "@solidjs/router";

export function NavigationMenuDemo() {
  return (
    <NavigationMenuItem>
      <NavigationMenuLink as={A} href="/docs">
        Documentation
      </NavigationMenuLink>
    </NavigationMenuItem>
  );
}
```
