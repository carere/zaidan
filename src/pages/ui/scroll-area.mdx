# Scroll Area

Augments native scroll functionality for custom, cross-browser styling.

## Installation

### CLI

```package-dlx
shadcn@latest add scroll-area
```

### Manual

Copy and paste the following code into your project.

```tsx file=../../registry/ui/scroll-area.tsx showLineNumbers

```

## Usage

```tsx
import { ScrollArea } from "~/components/ui/scroll-area";
```

```tsx showLineNumbers
<ScrollArea class="h-[200px] w-[350px] rounded-md border p-4">
  Jokester began sneaking into the castle in the middle of the night and
  leaving jokes all over the place: under the king's pillow, in his soup,
  even in the royal toilet...
</ScrollArea>
```

## Horizontal Scrolling

Use the `ScrollBar` component with `orientation="horizontal"` to enable horizontal scrolling.

```tsx
import { ScrollArea, ScrollBar } from "~/components/ui/scroll-area";
```

```tsx showLineNumbers
<ScrollArea class="w-96 whitespace-nowrap rounded-md border">
  <div class="flex w-max space-x-4 p-4">
    {/* Your horizontal content here */}
  </div>
  <ScrollBar orientation="horizontal" />
</ScrollArea>
```

## With Separator

```tsx
import { ScrollArea } from "~/components/ui/scroll-area";
import { Separator } from "~/components/ui/separator";
```

```tsx showLineNumbers
<ScrollArea class="h-72 w-48 rounded-md border">
  <div class="p-4">
    <h4 class="mb-4 text-sm font-medium leading-none">Tags</h4>
    <For each={tags}>
      {(tag) => (
        <>
          <div class="text-sm">{tag}</div>
          <Separator class="my-2" />
        </>
      )}
    </For>
  </div>
</ScrollArea>
```
