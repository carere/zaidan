# Item

A versatile component that you can use to display any content. Use it to display a title, description, and actions. Group it with the `ItemGroup` component to create a list of items.

## Installation

### CLI

```package-dlx
shadcn@latest add item
```

### Manual

Copy and paste the following code into your project.

```tsx file=../../registry/ui/item.tsx showLineNumbers

```

## Usage

```tsx
import {
  Item,
  ItemActions,
  ItemContent,
  ItemDescription,
  ItemFooter,
  ItemHeader,
  ItemMedia,
  ItemSeparator,
  ItemTitle,
} from "~/components/ui/item";
```

```tsx showLineNumbers
<Item>
  <ItemHeader>Item Header</ItemHeader>
  <ItemMedia variant="icon">
    <Icon />
  </ItemMedia>
  <ItemContent>
    <ItemTitle>Item Title</ItemTitle>
    <ItemDescription>Item description</ItemDescription>
  </ItemContent>
  <ItemActions>
    <Button>Action</Button>
  </ItemActions>
  <ItemFooter>Item Footer</ItemFooter>
</Item>
```

## API Reference

### Item

The main component for displaying content with media, title, description, and actions.

| Prop      | Type                                | Default     |
| --------- | ----------------------------------- | ----------- |
| `variant` | `"default" \| "outline" \| "muted"` | `"default"` |
| `size`    | `"default" \| "sm" \| "xs"`         | `"default"` |
| `as`      | `ValidComponent`                    | `"div"`     |
| `class`   | `string`                            |             |

You can use the `as` prop to render a custom element, for example a link:

```tsx showLineNumbers
<Item as="a" href="/dashboard">
  <ItemMedia variant="icon">
    <Home />
  </ItemMedia>
  <ItemContent>
    <ItemTitle>Dashboard</ItemTitle>
    <ItemDescription>Overview of your account and activity.</ItemDescription>
  </ItemContent>
</Item>
```

### ItemGroup

Container that groups related items together with consistent styling.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` |         |

```tsx
<ItemGroup>
  <Item />
  <ItemSeparator />
  <Item />
</ItemGroup>
```

### ItemSeparator

Separator that divides items in an item group.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` |         |

### ItemMedia

Display media content such as icons, images, or avatars.

| Prop      | Type                             | Default     |
| --------- | -------------------------------- | ----------- |
| `variant` | `"default" \| "icon" \| "image"` | `"default"` |
| `class`   | `string`                         |             |

```tsx
<ItemMedia variant="icon">
  <Icon />
</ItemMedia>
```

```tsx
<ItemMedia variant="image">
  <img src="..." alt="..." />
</ItemMedia>
```

### ItemDescription
