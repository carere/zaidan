# Sonner

An opinionated toast component for SolidJS.

## Installation

### CLI

```package-dlx
shadcn@latest add sonner
```

### Manual

Copy and paste the following code into your project.

```tsx file=../../registry/ui/sonner.tsx showLineNumbers

```

## Usage

Add the Toaster component to your root layout.

```tsx
import { Toaster } from "~/components/ui/sonner";

export default function RootLayout({ children }) {
  return (
    <>
      {children}
      <Toaster />
    </>
  );
}
```

Then you can use the `toast` function from anywhere in your app.

```tsx
import { toast } from "solid-sonner";
```

```tsx showLineNumbers
toast("Event has been created.");
```
