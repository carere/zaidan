# Input

Displays a form input field or a component that looks like an input field.

## Installation

### CLI

```package-dlx
shadcn@latest add input
```

### Manual

Copy and paste the following code into your project.

```tsx file=../../registry/ui/input.tsx showLineNumbers

```
