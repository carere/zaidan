# Empty

Use the Empty component to display an empty state.

## Installation

### CLI

```package-dlx
shadcn@latest add empty
```

### Manual

Copy and paste the following code into your project.

```tsx file=../../../registry/kobalte/ui/empty.tsx showLineNumbers

```
