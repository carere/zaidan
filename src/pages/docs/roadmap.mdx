# Roadmap

A look at what's coming next. These are the key milestones we're working toward to make Zaidan the definitive component toolkit for <SolidJS class="inline-block size-3.5 align-text-bottom mx-0.5" /> SolidJS.

<Steps>

### Sync Base UI Components

<InProgressBadge />

Port the <a class="font-medium underline-offset-4 underline" href="https://base-ui-docs-solid.vercel.app/" target="_blank">Base UI (SolidJs port)</a> version of each component and block into the registry. This will provide an alternative primitive layer alongside Kobalte, giving developers more choices for accessible foundations.

### Add RTL Support to Zaidan Agent

<PlannedBadge />

Add right-to-left (RTL) language support to the Zaidan agent, enabling it to generate and transform components with proper bidirectional text handling. This will ensure the registry serves developers building applications for RTL languages such as Arabic, Hebrew, and Persian.

### Expand Zaidan Agent

<UpcomingBadge />

Enhance the Zaidan agent's capabilities to support more usecases. This includes installing zaidan from the agent and improving its transformation capabilities by encoding more pattern recognition.

### Enrich the Registry with More Blocks and Components

<UpcomingBadge />

Expand the Zaidan registry with more blocks and components, covering a wider range of UI patterns and use cases. This will include both foundational primitives and higher-level components to meet the needs of diverse SolidJS applications.

</Steps>
