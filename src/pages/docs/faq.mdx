# FAQ

Frequently asked questions about Zaidan. This page will grow as the project evolves.

## Why does this project exist ?

- **A playground for Zaidan agent** — Zaidan is the best shadcn's registry for SolidJs components but it is also a testbed for iterating on AI-assisted component development workflows. Lot of improvements will come in the near future to push forward in this direcrtion.

- **A home for synced React components** — There are many great Shadcn based react components out there. Zaidan provides a place to port and host SolidJS versions of the best ones, keeping them maintained and accessible through a single registry.

## Why `Zaidan` ?

Zaidan is a Japanese word that means "foundation". We chose this name because we want Zaidan to be the foundation for building SolidJS applications with shadcn's design system. It also reflects our vision of creating a platform that empowers developers to create beautiful and functional user interfaces with ease.
