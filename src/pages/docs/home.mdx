## Why Zaidan?

We love SolidJS. Its fine-grained reactivity and simple mental model make it a joy to work with. Thus, in order to build polished, accessible UIs, we need a library of ready-to-use components that feel as professional as what React developers have with Shadcn/ui.

That's why we created Zaidan (which means "Foundation" in Japanese).

## What to Expect

Zaidan aims to bring everything you love about Shadcn/ui to the SolidJS ecosystem:

- **Shadcn Feature Parity** — We're working to port all Shadcn components. Same design, same API patterns, built for SolidJS from the ground up.
- **Community-Driven Priorities** — Not sure which component to build next? We listen. The most requested components get implemented first.
- **Flexible Installation** — Copy and paste the source code directly, or use the CLI for quick setup. Your codebase, your rules.
- **Accessible by Default** — Every component follows WAI-ARIA guidelines. Keyboard navigation, screen reader support, and focus management are built in, not bolted on.
- **Styled with Tailwind** — Works seamlessly with Tailwind CSS v4. Customize colors, spacing, and typography to match your brand.

## Roadmap

We have big plans for Zaidan. Here's what's coming:

- **RTL Support** — Full right-to-left language support for Arabic, Hebrew, and other RTL languages. We want Zaidan to work beautifully for everyone, everywhere.
- **Base-ui Flavor** — An alternative set of primitives for those who prefer a different foundation. More choices, more flexibility.
- **Claude Code Agents** — We will share with the community the agents we used to build and sync Zaidan with SolidJS. It's the best way to grow Zaidan available components list.
